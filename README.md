
# 🗼 Blog Paris - A Cidade Luz 🇫🇷

![Paris](https://images.unsplash.com/photo-1502602898657-3e91760cbb34)

## 📖 Sobre o Projeto

O **Blog Paris - A Cidade Luz** é um projeto desenvolvido para apresentar informações sobre uma das cidades mais famosas do mundo.

O site reúne conteúdos sobre a história de Paris, pontos turísticos, gastronomia, cultura, arte e curiosidades, criando uma experiência visual e informativa para os visitantes.

Além disso, o projeto possui um sistema de **curtidas com contador**, desenvolvido com JavaScript, permitindo que o usuário interaja com a página.

---

# ✨ Funcionalidades

O blog possui:

✅ Página informativa sobre Paris;  
✅ Layout moderno e responsivo;  
✅ Imagens ilustrativas;  
✅ Botão de curtida com contador ❤️;  
✅ História da cidade;  
✅ Principais pontos turísticos;  
✅ Comidas típicas francesas;  
✅ Curiosidades sobre Paris;  
✅ Informações culturais e artísticas.
